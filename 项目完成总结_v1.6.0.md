## 🚀 使用指南

### 快速开始（3分钟）

```bash
# 1. 克隆项目
git clone https://github.com/your-repo/jas-agent.git
cd jas-agent

# 2. 安装依赖
go mod tidy
cd web && npm install && cd ..

# 3. 启动服务（终端1）
cd cmd/server
go run main.go -apiKey sk-xxx -baseUrl https://api.openai.com/v1

# 4. 启动前端（终端2）
cd web
npm run dev

# 5. 访问 http://localhost:3000 🎉
```

### 功能测试

**测试1: ReAct Agent**
```
输入: 计算 15 + 27
预期: 返回 42
```

**测试2: Plan Agent**
```

预期: 显示执行计划，返回 64 磅
```

**测试3: 流式响应**
```
勾选"启用流式响应"
输入: 边境牧羊犬的平均体重是多少？
预期: 实时显示思考、查询、结果过程
```
